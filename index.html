About
Tacoma-JS promotes computer programming in the Javascript language by hosting meetups in Tacoma, WA with topics ranging from introductory to advanced, client side and server side, plus occasional mini hackathons emphasising collaboration using pair programming tools and methods, successful patterns, and best practices.
